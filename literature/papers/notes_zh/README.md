# RAG 评测论文中文精读笔记

本目录存放 RAG / knowledge-base 评测论文的中文精读笔记。

这些文件不是论文全文翻译，也不是可替代原文引用的译本；它们是面向本仓库实验设计和“公理化 LLM wiki”论文构思的结构化研究笔记。每篇笔记会尽量覆盖：

- 论文定位与研究问题
- 评测对象、任务设置、指标与实验设计
- 主要发现、局限与风险
- 对 `ragbench` 环境的实现启发
- 对“公理化 LLM wiki”论文的可借用点
- 可落地的测试项

原始 PDF 与英文 Markdown 见：

- [`../markdown`](../markdown)
